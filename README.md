# MediaForest

Personal media library application built on PhraseVault. Browse, search, and track your media library — share it with trusted friends.

## Getting Started

### First user (owner setup)

Open MediaForest in your browser. With no users registered, the login page shows "Set up this server". Choose a passphrase — this is your permanent identity. Your passphrase never leaves your browser; the server only stores your derived public key.

### Inviting additional users

As the owner, click **+ Invite** in the header to generate a one-time invite token (7-day expiry). Share the token with the person you want to add. They open the login page, click "Have an invite? Register instead →", enter their own passphrase and the token.

### Local auth agent (skip typing your passphrase)

Run the PhraseVault companion agent on your local machine. Once running, MediaForest detects it automatically and logs you in without a passphrase prompt.

See [github.com/christcb03/phrasevault#local-auth-agent](https://github.com/christcb03/phrasevault#local-auth-agent) for setup.

## Auth model

- Each user's identity is a secp256k1 keypair derived from their passphrase via BLAKE3. The passphrase never reaches the server.
- The first user to register becomes the owner. All subsequent registrations require an invite token generated by the owner.
- Watchlists are per-user. The media library (titles, files, sources) is shared across all users.
- Sessions expire after 24 hours.
